# 🌱 AI日記（無料版）

**毎晩、AIがその日を勝手にまとめて、日記と「絵日記カード」を作ってくれる仕組み。**
Claudeの有料プランなしで、GitHubの無料機能＋GoogleのAI（無料枠）だけで動きます。

- 📝 GitHubに「今日のメモ」を書くだけ
- 🤖 毎晩23時、AIが自動で **思考ログ・日記・絵日記** を生成
- 💾 全部このリポジトリに保存される（自分だけの記録棚）

---

## 仕組み（ざっくり）

```
📱 GitHubのIssueに今日のメモを書く
        ↓
⏰ 毎晩23時、GitHub Actionsが自動で起動（無料）
        ↓
🤖 Google Gemini が思考ログ・日記・絵日記を生成（無料枠）
        ↓
💾 thoughts/ と diary/ に保存される
```

---

## セットアップ（初回だけ・約15分）

### 1. このリポジトリを用意する
このリポジトリ（テンプレート）を自分のアカウントにコピーして使います。
すでに自分のリポジトリとして持っているなら、この手順はスキップでOK。

### 2. Google GeminiのAPIキーを取る（無料）
1. [Google AI Studio](https://aistudio.google.com/) を開く
2. 「Get API key（APIキーを取得）」をクリック
3. 表示されたキーをコピー（後で使います）

### 3. APIキーをリポジトリに登録する
1. このリポジトリの **Settings（⚙️）** を開く
2. 左メニュー **Secrets and variables → Actions**
3. **New repository secret** を押す
4. 次のとおり入力して保存：
   - **Name**: `GEMINI_API_KEY`
   - **Secret**: さっきコピーしたキー

> 🔐 キーはここに入れれば安全です（他人にも自分にも表示されません）。コードに直接書かないこと。

### 4. Actions を有効にする
リポジトリの **Actions** タブを開き、ワークフローの実行を許可（Enable）します。
※新しく作ったリポジトリだと最初は無効になっていることがあります。

これで準備完了です。

## 使い方（毎日）

### ① 今日のメモを書く
1. **Issues** タブ →「New issue」
2. テンプレート「今日の記録」を選ぶ（なければ普通に新規作成し、ラベルに `diary` を付ける）
3. 今日あったこと・考えたことを、時系列でダラダラ書くだけ
4. 追記したくなったら、そのIssueに**コメント**で書き足す

### ② 夜、勝手にまとまる
毎晩23時（日本時間）に自動で、
- `thoughts/年/月/日付.md` … 思考ログ
- `diary/年/月/日付.md` … 日記本文
- `diary/年/月/日付.svg` … 絵日記カード

が生成され、メモのIssueには「まとめました」というコメントが付いてクローズされます。

### すぐ試したいとき（手動実行）
**Actions** タブ →「毎日の日記まとめ」→「Run workflow」で、23時を待たずに実行できます。

## カスタマイズ

| やりたいこと | 方法 |
|---|---|
| まとめる時刻を変える | `.github/workflows/daily-diary.yml` の `cron` を変更（UTC表記。日本時間−9時間） |
| 使うAIモデルを変える | Settings → Secrets and variables → **Variables** に `GEMINI_MODEL` を追加（例: `gemini-2.0-flash`） |
| 絵日記のデザインを変える | `scripts/generate_diary.py` の `render_svg()` を編集 |
| メモに使うラベル名を変える | Variables に `DIARY_LABEL` を追加 |

## よくある質問

**Q. 本当に無料？**
GitHub（無料枠）とGemini（無料枠）の範囲で動きます。1日1回の実行なら、通常の無料枠に十分おさまります。

**Q. メモが無い日は？**
その日は何も生成されません（Issueが無ければ静かに終了）。

**Q. 記録は誰かに見られる？**
リポジトリを **Private（非公開）** にすれば、自分だけのものになります。
